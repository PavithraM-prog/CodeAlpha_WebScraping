# CodeAlpha Task 1 - Web Scraping

## Objective
This project demonstrates web scraping using Python and BeautifulSoup. Data was collected from the "Books to Scrape" website and exported to a CSV file.

## Technologies Used
- Python
- Google Colab
- Requests
- BeautifulSoup
- Pandas

## Website Used
https://books.toscrape.com/

## Data Extracted
- Book Title
- Price
- Availability
- Rating

## Output
The scraped data is stored in:

- books_data.csv
